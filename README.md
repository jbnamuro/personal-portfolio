🎨 Personal Portfolio
A modern, high-performance portfolio showcasing frontend expertise. Built with React and GSAP, featuring fluid transitions, full SEO optimization.

✨ Technologies

-React

-GSAP (Animations)

-Tailwind CSS

-Vite

🚀 Features
Smooth Motion: GSAP-powered page transitions and interactions.
SEO Ready: Complete Meta tags, JSON-LD schema, and automated sitemaps.

💡 The Process
I wanted to move away from static, boxy portfolios. Using GSAP, I focused on making the UI feel interesting and alive. The biggest hurdle was ensuring the heavy animations didn't tank performance on mobile devices. After significant optimization the result is a site that feels just as snappy on a phone as it does on a desktop.

🚦 Running the Project
Clone the repo

Install dependencies: npm install

Run dev server: npm run dev

Open: http://localhost:5173
